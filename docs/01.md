### 3. Module trong NestJS

**Module** là khối xây dựng chính của NestJS, giúp nhóm các thành phần liên quan như **Controller**, **Service**, **Provider**.

Ví dụ: Tạo module **User**:
Decorator @Module() cung cấp các thuộc tính chính để định nghĩa cấu trúc của module:
imports: Danh sách các module khác mà module này cần sử dụng.
controllers: Danh sách các HTTP controller sẽ được khởi tạo trong module.
providers: Danh sách các service được khởi tạo và có thể được inject (tiêm) vào các thành phần khác trong cùng một module.
exports: Danh sách các provider thuộc module này được phép chia sẻ cho các module khác đang import nó.

```bash
nest generate module user
```
### 4. Dependency Injection (DI) trong NestJS

NestJS sử dụng **Dependency Injection (DI)** để quản lý các service và provider một cách tự động.

Ví dụ: Inject **UserService** vào **UserController**:
Day la class controller
```javascript
import { Controller, Get } from '@nestjs/common';
import { UserService } from './user.service';

@Controller('user')
export class UserController {
  constructor(private readonly userService: UserService) {}

  @Get()
  findAll() {
    return this.userService.getAllUsers();
  }
}
```
Class service
```javascript
import { Injectable } from '@nestjs/common';

@Injectable()
export class UserService {
  getAllUsers() {
    return ['User 1', 'User 2', 'User 3'];
  }
}
```
